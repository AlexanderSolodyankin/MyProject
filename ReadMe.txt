Техническое задание по финальному проекту «Driver Room»
Студент группы JV-W21 Солодянкин Александр

Миссия и цели проекта:
1.	Платформа для поиска и обменом информации, с возможностями социальных сетей и создания форумов и групп интересов.
2.	Быстрый и удобный поиск сервис-центров и специалистов.
3.	Облегчение обмена услуг и товаров.
4.	 Помощь в решении технических задач по ремонту машин при помощи форумов.
5.	Социальная сеть для обмена навыками между водителями.
Функционал:
1.	Регистрация и авторизация.
2.	Регистрация собственных  предприятий по сервисному обслуживанию машин.
3.	Регистрация в качестве специалиста по предоставлению частных услуг по ремонту или консультации.
4.	Создание форумов для обсуждения разного рода проблем.
5.	Переписка в локальном чате или в общем.
6.	Объявление публикации или последних новостей а также рекламная публикация для развития бизнеса.
7.	Отзывы о сервис центрах или специалистов в качестве контроля качества обслуживания.
8.	Фильтрация публикации по интересам и подписки.
9.	Создание сообществ интересов.
10.	Редактирование личной информации форумов и публикаций.
11.	Сброс пароля пользователя через почту.
12.	Онлайн чат в реальном времени и возможность создание онлайн чатов по интересам для обсуждения разного рода вопросов и переписки в реальном времени.

Технические требование к проекту:
PostgreSQL, Spring Boot, Spring Data, Spring Security, Lombok, Maven, Swagger.

Задачи по построению проекта:
1.	Построить архитектуру REST API приложения.
2.	Реализация пользователя и обеспечение защиты путем шифрования.
3.	Подтверждение регистрации пользователя через почту и редактирование личной информации
4.	Реализация возможности регистрации сервис центров или специальностей и возможность их редактирования.
5.	Реализация публикации и комментирование публикации.
6.	Реализация возможности подписок а так же создание групп интересов
7.	Реализация возможности добавления друзей
8.	Реализация возможности комментирования сервис центров и бронирования мест.
9.	Личная переписка между пользователями и специалистами
10.	Онлайн чат в реальном времени и возможность создавать группы онлайн чатов
11.	Реализация торговой площадки для рекламирования услуг и товаров.
12.	Добавить возможность добавление файлов и фотографий

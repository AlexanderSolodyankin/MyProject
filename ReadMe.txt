DriveRoom – Площадка для общества водителей
Автор: Солодянкин Александр (JV-W21)

Цель:

•	Площадка для автолюбителей, сервис центров а так же специалистов по данной тематике.
•	Обеспечить быстрый и легкий доступ актуальной информации и связь с специалистами.
•	Создание упрощенной социальной сети автолюбителей.
Функционал
•	Авторизация
•	Аутентификация
•	Добавление личной инфы
•	Регистрация сервис центра
•	Добавление специальности
•	Добавление поста, коментариев к посту.
•	Личный чат переписка
•	Вывод сервис центров
•	Поиск сервис центров
•	Вывод специалистов
•	Поиск специалистов или сервис центров
